# 프로젝트 개요
본 프로젝트는 온디바이스 AI 기반 보안 앱 서비스 개발을 목표로 합니다.  
개인정보 유출 및 피싱과 같은 사회적 문제를 해결하기 위해,  
민감한 데이터를 클라우드에 전송하지 않고 사용자 기기 내부(On-Device)에서  
실시간으로 위협을 탐지하고 대응하는 AI 서비스를 구현합니다.
